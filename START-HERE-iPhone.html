ScoreKit iPhone Package

Install ScoreKit on iPhone

This folder contains the ScoreKit prototype: a black-and-white game scoring app with groups, organizer/player roles, match history, and all-time win/loss tracking.

Quick Preview

On a computer, open INSTALLER.html or index.html to preview the app in a browser.

Open Installer Open ScoreKit

Install On iPhone

To install it like an app on your iPhone Home Screen, the folder must be uploaded to an HTTPS website first.

  1. Upload this entire folder to Netlify, Vercel, Cloudflare Pages, GitHub Pages, or any HTTPS web host.
  2. Open the hosted INSTALLER.html URL in Safari on your iPhone.
  3. Tap the Safari Share button.
  4. Tap Add to Home Screen.
  5. Open ScoreKit from your iPhone Home Screen.

What Is Included

  • Basketball, pickleball, and badminton scoring
  • Groups for game organizers
  • Organizer, Player, and Organizer + Player roles
  • During-game win/loss projection
  • All-time win/loss records per active group
  • Offline-ready PWA files

Important

If you only send the zip file to your iPhone, you can store it in Files, but iOS will not install it as a Home Screen app until it is opened from an HTTPS website in Safari.
