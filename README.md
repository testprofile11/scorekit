# J1 Hoops Scoreboard

Simple basketball-only scoreboard for J1 Hoops.

It includes only:

- Team A name
- Team B name
- Game scores
- 12-minute timer
- Editable timer
- Start / pause / reset controls

No groups, no player records, no login, no match history.

## GitHub Pages

Upload all files in this folder to a GitHub repository and enable GitHub Pages from the `main` branch, `/root`.
